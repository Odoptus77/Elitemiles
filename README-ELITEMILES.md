# EliteMiles (MVP)

## Local dev

```bash
cd elitemiles
npm run dev
```

Open http://localhost:3000

## Deployment (free)

Recommended: Vercel (free tier)
- Import this repo/folder
- Framework preset: Next.js
- Build command: `npm run build`
- Output: Next.js

## MVP scope
- Learn hub (MDX)
- Deals (manual entries + filters)
- Programs DB (seed + search)
- Tools: redemption value calculator + transfer partner optimizer (basic)
